# Changelog

*This document will contain any changes made to public releases.

* Update 0.4.4:

Olivine Battle Shop open at the Olivine Mart. 
Add an NPC to the Second-Class Cabin to access a PC for trading. 
More hints at Feraligatr Lair.

Another way to awaken Ancestor Meganium.
